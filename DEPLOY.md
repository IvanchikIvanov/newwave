# Инструкция по деплою

Проект готов к деплою на любую платформу. Поддержка мобильных устройств уже добавлена.

## Быстрый деплой (рекомендуется)

### 1. Vercel (самый простой)

1. Зарегистрируйся на [vercel.com](https://vercel.com)
2. Подключи GitHub репозиторий
3. Vercel автоматически определит настройки из `vercel.json`
4. Нажми Deploy

**Готово!** Сайт будет доступен по адресу `твой-проект.vercel.app`

### 2. Netlify

1. Зарегистрируйся на [netlify.com](https://netlify.com)
2. Подключи GitHub репозиторий
3. Настройки из `netlify.toml` применятся автоматически
4. Нажми Deploy

**Готово!** Сайт будет доступен по адресу `твой-проект.netlify.app`

### 3. GitHub Pages

1. Залей код в GitHub репозиторий
2. Перейди в Settings → Pages
3. Source: GitHub Actions
4. Workflow из `.github/workflows/deploy.yml` запустится автоматически

**Примечание:** Если репозиторий называется не `gungeon-lite-boss-rush`, измени `base` в `vite.config.ts`:
```typescript
base: '/имя-твоего-репозитория/',
```

## Локальная сборка для проверки

```bash
npm run build
npm run preview
```

## Что было добавлено для мобильных:

- ✅ Touch события для управления (один палец = атака, два пальца = бомба)
- ✅ Адаптивный интерфейс с подсказками для мобильных
- ✅ Viewport meta тег для правильного масштабирования
- ✅ CSS класс `touch-none` для предотвращения стандартных жестов

## Проверка мобильной версии

После деплоя открой сайт на телефоне или используй DevTools браузера (F12 → Toggle device toolbar).
